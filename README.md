# AI Comment Analytics & Moderation System

## Description
This project is an AI-based comment analysis system that classifies user comments into toxic and positive categories using machine learning. It identifies harmful content such as insult, hate, threat, and harassment, along with positive intent like love and support.

## Objective
To automate the analysis of large volumes of user comments and assist content moderation by detecting toxic patterns efficiently.

## Approach
Text data is cleaned and preprocessed using NLP techniques. Comments are converted into numerical features using TF-IDF, and a multi-label machine learning model is used for classification. The results are displayed through a simple web interface.

## Technologies Used
Python, Scikit-learn, NLTK, Streamlit

## Outcome
The system predicts one or more labels for a given comment and supports moderation and analytical decision-making.
