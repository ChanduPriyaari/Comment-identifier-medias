
This project analyzes user-generated comments using Machine Learning to
identify toxic behavior such as insult, hate, threat, and harassment,
as well as positive intent like love and support.

## Features
- Multi-label comment classification
- Batch and single comment analysis
- Analytics dashboard using Streamlit

## Tech Stack
- Python
- Scikit-learn
- NLTK
- Streamlit

## Use Case
Helps in moderating online communities by automatically analyzing
large volumes of comments.
